# 🏥 Hospital OPD Wait Time Predictor

A machine learning web app that predicts OPD waiting times at Indian government hospitals.

## 🌐 Live Demo
[👉 Click here to open the live app](https://hospital-queue-predictor.onrender.com)

## 📌 Pages
- **/** — Main prediction page
- **/heatmap** — Weekly wait time heatmap
- **/compare** — ML model comparison (RF vs DT vs LR)

## 🧠 ML Model
| Detail | Value |
|---|---|
| Algorithm | Random Forest Regressor |
| Training Rows | 8,000 |
| R² Score | 0.9254 |
| MAE | ~5 minutes |
